# Nuxt3 ランディングページ

モダンで美しいランディングページのテンプレートです。

## 特徴

- 🎨 モダンで洗練されたデザイン
- 📱 完全レスポンシブ対応
- ⚡ Nuxt3 + Vue3 + TypeScript
- 🎭 Tailwind CSS によるスタイリング
- 🌈 グラデーションとアニメーション効果
- 📦 すぐに使える各種セクション

## セットアップ

### 依存関係のインストール

```bash
npm install
```

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:3000` を開いてください。

### プロダクションビルド

```bash
npm run build
```

### 静的サイト生成

```bash
npm run generate
```

## 含まれているセクション

- ヒーローセクション（メインビジュアル）
- 機能紹介セクション
- 統計情報セクション
- 料金プランセクション
- CTAセクション
- フッター

## カスタマイズ

- `pages/index.vue` - メインのランディングページ
- `assets/css/main.css` - グローバルスタイル
- `nuxt.config.ts` - Nuxt設定
